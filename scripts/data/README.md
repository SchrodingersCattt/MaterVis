# Bundled example structures

| File | Summary | License |
| --- | --- | --- |
| `DAP-4.cif` | Ordered diammonium diperchlorate polymorph `[C₄H₁₂N₂][ClO₄]₂`, Z = 12, triclinic `P1`, a = b = c ≈ 14.4264 Å, α = β = γ = 90°. Generated by Materials Studio (`_audit_creation_date 2024-08-27`). | Ships with `crystal_viewer` for demonstration only — please cite the originating publication if you use it in downstream work. |

Element counts (per cell): C=48, H=144, N=24, Cl=24, O=96. The ClO₄⁻ framework is paired with twelve C₄H₁₂N₂²⁺ dications.

Add your own CIFs to this folder and reference them from the example scripts;
the viewer accepts any CIF that [`gemmi`](https://gemmi.readthedocs.io/) can
parse, including Materials-Studio exports that drop the optional
`_atom_site_disorder_*` columns.

A larger curated catalog can be wired into the Dash viewer by writing
`catalog.local.json` at the repo root (see `AGENTS.md`).
